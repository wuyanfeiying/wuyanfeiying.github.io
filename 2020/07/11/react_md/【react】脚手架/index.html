为何要用脚手架

现如今，web端开发项目结构复杂，并非只用单页面文件就可以完成。目前比较流程的做法就是采用Webpack + ES6模式来搭建项目构架，然后通过打包的方式发布web应用。
一般业内称这个项目构架“脚手架”，就如同盖房子的时候必需使用脚手架一样。

react脚手架一些涉及点如下

react 官方推荐使用的脚手架： create-react-app

  1. Node.js框架
  2. npm包管理工具
  3. Webpack模块打包器
  4. ES6(ECMAScript 6)
  5. Babel 编译器
