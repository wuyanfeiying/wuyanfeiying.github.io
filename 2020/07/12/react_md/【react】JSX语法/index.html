【react】React JSX语法

发表于 2020-07-12 分类于 react

JSX 是React框架内置的语法，且专用于React应用开发，开发人员使用JSX方式来实现UI中的虚拟DOM。

  • 在React项目中使用JSX语法，必需引用bacel.js来解析JSX。
  • 在使用type='text/babel'属性替换type='text/javascript'属性后，浏览器内置的JavaScript解析器就不会解析<script>标签里的脚本代码，转而由“babel.js”进行解析，从而避免React代码与原生JavaScript代码产生混淆。

JSX一般语法形式

  • const 定义常量element
  • 标签元素表示DOM标签，且支持多级嵌套
