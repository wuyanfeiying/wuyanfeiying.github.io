【react】React组件（介绍/函数组件/类组件/组合组件）

发表于 2020-07-12 分类于 react

介绍

React组件可以将UI切分成一些独立的、可复用的部件，这样有助于设计人员专注于构建每一个单独的部件。
React组件通过Props可以接受任意的输入值，因此Props可以理解为参数逇概念。

React组件从形式上来看很像是JavaScript函数，通过这个函数返回一个需要在页面上展示的React元素。

特征

  • 结构看起来十分简洁和高效
  • 在函数内部定义好页面需要的元素组合，通过 return 语句返回函数值就可以。

通过ES6 Class 来实现

由于React语法是基于版本 ECMAScript 6 实现的。因此，React组件除了通过JavaScript函数的形式，还可以通过 ES6 Class(类)的形式来实现。

React类组件

  • 通过class关键字定义了类名，并通过extend关键字声明该类继承自React.Component对象。
  • 使用React.Component对象的render()方法实现React类组件的渲染操作。

React组合组件

React组件支持在自身定义中引用其他组件，从而构成React组合组件。

  • 优点：可以使用同一组件来抽象出任意层次的细节
