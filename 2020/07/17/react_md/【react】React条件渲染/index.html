【react】React条件渲染

发表于 2020-07-17 分类于 react

介绍

  • 在React框架中,设计人员可以创建不同组件来封装各种业务需求,然后依据需求的不同状态,仅仅渲染组件中对应状态下的局部内容。
  • React条件渲染基本上是通过JavaScript条件运算符(if语句,与逻辑,三元表达式等等)创建元素组件,并让react根据组件的不同状态渲染更新UI来完成。

整理一些没注意的小细节

为什么逻辑与（&&）运算可以实现条件渲染的效果？

因为在JavaScript语法中，表达式（true&&expression）总会返回expression,而表达式（false&&expression）则总会返回false,因而可以实现React条件渲染功能。

如何阻止组件渲染

在一些特殊场景下，设计人员可能需要隐藏特定的组件，即使这些组件已经被渲染完成了。如果要完成隐藏组件的操作，React允许通过render()方法直接空(null)，进而达到不进行任何渲染的功能。

  • 在React组件的render（）方法中返回空（null），并不会影响组件的生命周期。
