【react】虚拟DOM

发表于 2020-07-11 更新于 2020-07-12 分类于 react

React中引入了虚拟DOM的概念，在设计传统的HTML网页UI的时候，通常需要通过实际的创建和删减DOM元素，来更改页面的外观和数据变化。对于复杂的页面来说，频繁的操作DOM元素，往往会使页面性能降低，从而导致用户使用感受变差。

为此，React框架引入了虚拟DOM机制，React DOM类似于一种将相关的实际DOM组合在一起的集合，是有区别于传统概念上的DOM元素，理解为 DOM组件也尚可。
